# LoRA Downloader for ComfyUI

A custom node plugin that allows you to download, organize, and manage LoRA models directly inside ComfyUI.

## Features

- Download LoRA models from URLs (Civitai and direct links supported)
- Manage local LoRA directories
- Automatic naming & file organization
- Progress display in UI

## Installation

### Using ComfyUI-Manager
Search for `LoRA Downloader for ComfyUI` and install.

### Manual install
```bash
cd ComfyUI/custom_nodes
git clone https://github.com/huihuihuiz/lora_downloader
